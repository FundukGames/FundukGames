What is Hashi?

Hashi — short for Hashiwokakero ("build bridges"), also known simply as Bridges — is a classic Japanese logic puzzle. Each circle is an island, and its number says how many bridges reach it. Bridges run horizontally or vertically, at most two between any pair, and may never cross. When every number is satisfied and all islands form one connected network, the puzzle is solved.

Every board here is generated so the whole network can be deduced with counting logic alone — one unique solution, no guessing. Three board sizes, unlimited play, and a shared Daily Challenge with streaks.

The rules

  1. Tap between two islands to add a bridge; tap again for a double bridge, again to remove.
  2. Bridges run straight (horizontal or vertical) and never cross each other or islands.
  3. Each island's number = its bridge count (islands turn amber when exact).
  4. The finished network must connect every island — no separate groups.

Frequently asked questions

Is Hashi free?
Yes — completely free, supported by ads. No download, no login.
Do I ever need to guess?
No — every board is verified solvable by forced counting moves alone, and each has exactly one solution.
Can bridges run diagonally?
No — horizontal and vertical only, and only between islands with a clear line between them.
Is there a daily puzzle?
Yes — a shared daily board with streaks, plus unlimited random boards in three sizes.

Quick solving tips

  • Corners and edges first. An island with few visible neighbors is tightly constrained — a corner "4" or an edge "6" is forced instantly.
  • Count the capacity. If an island's number equals everything its neighbors can accept, max out every link.
  • The all-but-one rule. If ignoring one neighbor leaves too little capacity, at least one bridge to that neighbor is guaranteed.
  • Watch for cut-offs. Don't close a small group into a private network — everything must join up at the end.
